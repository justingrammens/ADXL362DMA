Detailed Description

template<size_t BUF_SIZE>
class ADXL362DataEx< BUF_SIZE >

Class used to store data from the FIFO.

Templated version that allows the buffer size to be specified. The ADXL362 supports up to 511 samples of either 6 bytes (no temperature) or 8 bytes (with temperature), so the buffer could be up to 512 * 8 = 4096 bytes.
